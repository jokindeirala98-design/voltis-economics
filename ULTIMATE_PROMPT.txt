## 1. PROMPT DE SISTEMA PARA IA (CONTEXTO INICIAL)
"Actúa como un Ingeniero Full-Stack Senior. Tu misión es crear una aplicación de análisis de facturas eléctricas (PDF/Excel) de alta gama. El diseño debe ser 'Dark Glassmorphism' (Fondo #020617, tarjetas translúcidas, luces radiales, tipografía Inter). Usa Next.js 15, Tailwind v4 y Lucide React. La funcionalidad principal es extraer datos de facturas mediante Gemini AI, permitir la edición manual en una tabla transpuesta y generar reportes visuales avanzados (Recharts) con descarga en PDF (react-to-print)."

### B. Extractor IA (src/lib/gemini-extractor.ts)
```typescript
import { GoogleGenerativeAI } from '@google/generative-ai';

const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY || '');

const SYSTEM_PROMPT = `
Eres un analista experto en facturación eléctrica de España. Devuelve ÚNICAMENTE un JSON con:
1. Periodos P1-P6 extraídos fielmente.
2. Agrupación en otrosConceptos: 'Bono Social', 'Alquiler de equipos', 'Impuesto Eléctrico', 'IVA / IGIC'.
3. 'Peajes y Transportes' solo si se cobran aparte.
4. Cuadre matemático: costeTotalConsumo + costeTotalPotencia + otrosConceptos = totalFactura.
Devuelve el JSON plano sin backticks.
`;

export async function extractBillDataWithAI(pdfText: string) {
  const model = genAI.getGenerativeModel({ 
    model: 'gemini-flash-latest',
    generationConfig: { responseMimeType: "application/json" }
  });
  const prompt = `${SYSTEM_PROMPT}\n\nFACTURA:\n${pdfText}`;
  try {
    const result = await model.generateContent(prompt);
    let output = result.response.text().trim();
    const match = output.match(/\{[\s\S]*\}/);
    if (match) output = match[0];
    return JSON.parse(output);
  } catch (error) { throw error; }
}
```

### NOTA FINAL PARA ANTIGRAVITY:
Asegúrate de que el Reporte Visual tenga un `selectedBillId` para los modales que use el `id` único de la fila (`row.id`) y no el índice del array, para evitar errores de desincronización al ordenar datos. Los números en el gráfico de barras deben estar formateados con `toFixed(2)` en los tooltips.
